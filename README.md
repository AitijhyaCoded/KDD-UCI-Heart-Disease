# Data Mining & Data Preprocessing for Knowledge Discovery

This mini project demonstrates the **early stages of the Knowledge Discovery in Databases (KDD) process** using the **UCI Heart Disease Dataset**. The project focuses on transforming raw healthcare data into **machine-learning ready data** through systematic analysis and preprocessing.

## Tools

Python, Pandas, NumPy, Matplotlib, Scikit-learn

## Key Tasks

* Understanding **data objects and attribute types**
* Performing **Exploratory Data Analysis (EDA)**
* Computing **correlation and similarity measures**
* Assessing **data quality and missing values**
* Applying **imputation techniques**
* Building a **preprocessing pipeline** for ML

## Pipeline

```
SimpleImputer → Scaling → LogisticRegression
```

## Outcome

The project highlights how **data preprocessing and quality assessment are crucial for successful machine learning and knowledge discovery**.
